Node.js

Node.js is a JavaScript runtime built on Google Chrome's V8 engine. It utilizes an event-driven, non-blocking I/O model that makes it lightweight and efficient, the perfect fit for a raspberry project. Since my studies mainly surrounds the usage of node.js in a web development sphere it was the clear chooice.
